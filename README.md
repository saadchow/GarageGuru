## About The Project 📝

<p align="center">
  <img src="https://i.imgur.com/SfeXfNU.png" alt="GarageGuru screenshot">
</p>

**GarageGuru** is a social web app for automotive hobbyists. Share your builds, browse and search projects from other enthusiasts, and connect without the usual noise—just clean, car-focused content.

- **Hosting:** Render ([live app](https://garageguru.onrender.com/))  
- **Media storage:** Cloudinary (replaces Amazon S3)
